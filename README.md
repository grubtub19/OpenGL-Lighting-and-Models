## Features

* Omni-directional shadow mapping
* Blinn-Phong shading
* .Obj model importer
* Camera control (w a s d q e)
* Light-source controls (↑ ↓ → ← < >)
* Example Enviroment
  * Custom-posed and texture-editted models via Blender
